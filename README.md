# Como Rodar?

## Requisitos

* Node 20+

## Clonando Repositório
Para clonar o repositório, no seu prompt, digite:
```bash
git clone https://github.com/C-WD/portfolio_frontend.git
```

## Instalando Dependências

Para instalar as dependências, basta rodar o comando:
```bash
npm install
```
